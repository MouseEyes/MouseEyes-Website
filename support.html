MouseEyes Support

MouseEyes is designed to be lightweight, simple, and easy to use. If you need help, have a question, or want to report an issue, support is available.

Getting Help

The best way to get support is by email.

Email: Admin@MouseEyes.com

Please include:

  • A brief description of the issue or question
  • Your Windows version (Windows 11)
  • Any relevant details about your monitor setup

Common Topics

  • Installation questions
  • Multi‑monitor behavior
  • Profile usage and customization
  • Off‑screen window recovery
  • Licensing questions

Response Time

Support requests are typically answered within a reasonable time. MouseEyes is an independently developed project, and responses may not be immediate, but all legitimate inquiries are read.
